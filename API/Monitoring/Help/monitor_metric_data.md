### 3. lastDownsample

**查看某资源的最后一个监控数据点**

请求示例

```
https://monitor.jdcloud-api.com/v1/regions/cn-north-1/metrics/memory.usage/lastDownsample?timeInterval=1h&serviceCode=vm&resourceId=i-test01
```

返回

```
{
    "requestId": "345b7934-f178-46a2-bfc8-786e8af363fb",
    "result": {
        "items": [
            {
                "metric": "memory.usage",
                "name": "i-test01",
                "value": 4
            }
        ]
    }
}
```
