# disableSubUserAccessKey


## 描述
禁用子用户的AccessKey

## 请求方式
PUT

## 请求地址
https://iam.jdcloud-api.com/v1/subUser/{subUser}/accessKey/{accessKey}:disable

|名称|类型|是否必需|默认值|描述|
|---|---|---|---|---|
|**subUser**|String|True| |子用户名称|
|**accessKey**|String|True| |accessKey|

## 请求参数
无


## 返回参数
|名称|类型|描述|
|---|---|---|
|**requestId**|String| |


## 返回码
|返回码|描述|
|---|---|
|**200**|OK|
