# 京东云DDoS基础防护相关接口


## 简介
京东云DDoS基础防护相关接口


### 版本
v1


## API
|接口名称|请求方式|功能描述|
|---|---|---|
|**describeIpResourceFlow**|GET|查询公网Ip的监控流量|
|**describeIpResourceInfo**|GET|查询公网Ip基本信息|
|**describeIpResourceProtectInfo**|GET|查询公网Ip的防护明细|
|**describeIpResources**|GET|查询区域下的公网Ip资源列表|
|**setCleanThreshold**|POST|设置公网Ip的清洗阈值|
