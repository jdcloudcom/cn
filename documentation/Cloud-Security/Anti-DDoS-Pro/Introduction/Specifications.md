# 产品规格

## 实例规格列表

| 规格列表 |  BGP线路防护能力 |  
| :------ |:---------: |
| IP数量    |  1 到 50个 |
| 保底防护  | 10G 到 600G  |
| 弹性防护  | 0G 到 1100G  |  	
| 业务带宽  | 100M 到 2000M  |

| 防护域名  |  60 到 20000个  | 

最大防护能力达1.5T。
