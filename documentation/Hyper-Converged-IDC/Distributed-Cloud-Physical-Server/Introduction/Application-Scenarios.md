# 应用场景
## 视频监控
政府、安防、商业等多类场景视频监控需求，视频流众多，存储容量需求大，直接上云带宽成本高、质量服务保障难度大。分布式云物理服务器可提供同城同区就近高性能计算，高性能存储服务器，解决视频收流、存储、智能分析，实现低成本、高质量的智能监控、存储。
